CS50 Final Project

Thai Vu Duc Anh

FocusFlow is a comprehensive study planner app that helps users manage tasks effectively.

Technologies and Features:

  • Building a web app using Flask
  • Website layout and design using HTML5, Jinja2, and CSS
  • Login and Logout functionalities using Flask-Session and Werkzeug
  • Database manipulation using SQLite
  • Task management functionalities with CRUD operations
  • Interactive and responsive design with custom CSS
  • Google Calendar integration using the Google API (coming soon)

How to Use FocusFlow:

  1. Register or Log In: Create a new account or log in to save and manage your tasks.
  2. Input Your Tasks: Add tasks in the Task Manager section with details like deadlines & estimated time.
  3. Track Your Current Tasks: View and manage your tasks dynamically in the Task List.
  4. Use the Pomodoro Timer: Stay focused by timing work sessions & breaks directly from Dashboard.
  5. Monitor Your Progress: Use the Pomodoro Timer to manage your time spent on tasks and breaks.
  6. Visualize Your Schedule: Use the Calendar to plan tasks effectively and avoid overlapping deadlines.
  7. Update and Delete Tasks: Edit or remove tasks to keep your workflow up-to-date.
  8. Optimize Your Productivity: Utilize filters, priorities, and categories to focus on what's important.
  9. Review Progress: Analyze remaining time & completed tasks to plan for upcoming weeks efficiently.
